# Pinout List
Raspberry Pi Pico WH Pinout - The Number is the header number, not GPIO Number
Pin 20 - CS7
Pin 21 - MISO
Pin 22 - CS0
Pin 24 - SCK
Pin 25 - MOSI
Pin 26 - CS1
Pin 27 - CS2
Pin 29 - CS3
Pin 31 - CS4
Pin 32 - CS5
Pin 34 - CS6

Every PCB shares the MOSI, MISO, and SCK lines.
